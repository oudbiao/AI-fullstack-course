## 📚 网站介绍

**AI 全栈学习教程** 是一套面向零基础学习者的系统化教程，从 Python 编程、数据分析与可视化、数学基础，到机器学习、深度学习、大语言模型与 AI Agent 开发，覆盖从入门到应用的完整技术栈。

### 适合谁

- 想转行或入门 AI / 数据 / 开发的零基础学习者  
- 已有编程基础、希望系统补全 AI 全栈能力的学习者  
- 希望按「路线图 + 项目」自学、构建作品集的求职者  

### 课程特点

- **分阶段设计**：第零阶段（开发工具）→ 第一至第五阶段（Python、数据、数学、机器学习、深度学习）→ 第六至第十阶段（CV、NLP、大模型、Agent、AIGC 等），可按职业方向选学  
- **理论 + 实战**：每阶段配有知识点讲解与动手项目（如房价预测、客户流失、用户分群、Kaggle、图像分类等）  
- **开源免费**：全部内容在 GitHub 开源，可在线阅读或本地构建  

### 主要内容概览

| 阶段 | 主题 |
|------|------|
| 第零～一阶段 | 开发工具、Python 编程基础 |
| 第二～三阶段 | 数据分析与可视化、AI 数学基础（线性代数、概率、微积分） |
| 第四阶段 | 机器学习（监督/无监督、特征工程、评估与调参） |
| 第五阶段 | 深度学习基础（神经网络、PyTorch、CNN/RNN/Transformer） |
| 第六～十阶段 | 计算机视觉、NLP、大模型原理与微调、大模型应用、AI Agent、AIGC 等 |

更多阶段说明、学习路线与职业方向，请访问 **[learning.airoads.org](https://learning.airoads.org)** 查看完整课程结构。
