## 一、这个模块在解决什么问题？

### 1.1 模块定位

经典机器学习补充 不是为了“多学一点内容”，而是为了补上某类经常决定工程上限的能力。

你可以先把它理解成一组专题工具箱：

- 遇到相关项目时再回来看
- 不必一次学完
- 但一旦进入对应场景，它们会非常有价值

### 1.2 建议学习顺序

比较稳妥的学习方式通常是：

1. 先看总览，知道每节课大概做什么
2. 先做最基础、最能马上派上用场的主题
3. 再进入更偏工程或更偏项目的内容

## 三、一个最小可运行示例

:::info 运行提示
```bash
pip install numpy scikit-learn
```
:::

```python
import numpy as np
from sklearn.svm import SVC

X = np.array([[0, 0], [1, 1], [1, 0], [0, 1]])
y = np.array([0, 1, 1, 0])

clf = SVC(kernel="linear")
clf.fit(X, y)
print(clf.predict([[0.8, 0.9]]))
```

### 3.2 这个示例想让你抓住什么？

这一小段代码不是为了覆盖整个模块，而是为了帮你迅速建立一个“这个模块到底在做什么”的感觉。

阅读时优先看三件事：

- 输入是什么
- 中间过程发生了什么
- 输出如何对应到实际项目
