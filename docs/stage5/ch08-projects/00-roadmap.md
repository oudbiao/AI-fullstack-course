# 学前导读：项目实战这一章到底该怎么学

这一章不是继续堆概念，而是把前面学过的训练闭环真正做成项目。

## 三个项目分别在练什么

| 项目 | 任务类型 | 你真正要练什么 |
|---|---|---|
| 图像分类 | CNN 项目 | 从训练到评估的完整图像任务闭环 |
| 文本情感分析 | 文本分类项目 | 标签设计、baseline、错误分析和升级路线 |
| 生成模型实战 | 生成项目 | 质量、多样性、稳定性和展示框架 |
