# 第二阶段：数据分析与可视化

| 信息 | 说明 |
|---|---|
| **预估学时** | 80～120h |
| **前置要求** | 完成第一阶段 |

掌握数据处理、分析与可视化的核心工具与方法。

## 为什么这个阶段很重要？

AI 的一切都始于**数据**。无论你未来做机器学习、深度学习还是大模型应用，第一步永远是：获取数据 → 理解数据 → 清洗数据 → 分析数据。

本阶段你将掌握 Python 数据科学的"三件套"：**NumPy**（数值计算）、**Pandas**（数据处理）、**Matplotlib/Seaborn**（可视化），这三个库在 AI 领域无处不在。

## 学习建议

- **先做预热**：第 1 章的预热练习一定要做，它会让你后续学习事半功倍
- **多用 Jupyter**：数据分析非常适合在 Jupyter Notebook 中进行，边写边看结果
- **跟着敲代码**：每个示例都亲手运行一遍，修改参数看看效果有什么变化
- **用真实数据**：学完基础后，找一个你感兴趣的数据集（Kaggle 上有很多）来练手

## 阶段导读

这一阶段要建立的不是“会几个库”，而是数据工作流意识：

1. 数据是怎么读进来的
2. 数据是怎么被清洗和转换的
3. 结果是怎么被统计和可视化的
4. 一份分析结论是怎么被表达出来的

后面做机器学习、深度学习和 RAG 时，你会不断回到这里用到这些能力。

## 建议学习顺序

1. 先做第 1 章预热  
   先体会纯 Python 处理数据为什么痛苦。

2. 再完整学习 NumPy  
   把数组、shape、广播和向量化打稳。

3. 然后学习 Pandas  
   把清洗、筛选、聚合、合并打顺。

4. 再学可视化  
   把结论表达清楚。

5. 最后按需补数据库和项目章  
   把数据从“代码练习”推进到“真实分析”。

## 学这一阶段最容易犯的错

- 只记 API，不理解数据流
- 图画出来了，但不知道图想表达什么
- 清洗时随手删数据，却不记录依据
- 只看 Notebook 输出，不总结结论

## 学完后的出口能力

- 能独立读取、清洗、分析和可视化中小规模数据
- 能看懂后续模型训练前的数据准备代码
- 能完成一次最小探索性数据分析
- 能把分析结果整理成图表和简短结论
