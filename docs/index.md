# AI 全栈学习教程

这不是一组松散的文章，而是一条从零基础走到 AI 应用、LLM 工程和 Agent 系统的完整学习路径。

## 这套课适合谁

- 想从零开始系统学 AI 的新人
- 已经会一点代码，但想把机器学习、深度学习和大模型主线串起来的人
- 想做 RAG、Agent、AI 应用项目，但基础不够扎实的人
- 想把“学过很多知识点”变成“能完成完整项目”的人

## 你会在这里学到什么

这套课覆盖 4 条逐步展开的主线：

- 编程与数据基础：终端、Git、Python、数据分析、数学、机器学习
- 模型主线：深度学习、CV、NLP、Transformer、LLM 原理
- 应用工程主线：RAG、应用开发、工程化、部署、日志与评估
- 智能体主线：推理、工具、记忆、MCP、多 Agent、生产化

## 三条推荐学习路线

### 路线一：从零到 AI 应用开发

适合大多数新人：

`第零阶段 -> 第一阶段 -> 第二阶段 -> 第三阶段 -> 第四阶段 -> 第五阶段 -> 第八B阶段 -> 第九阶段`

这条路线的重点是：先把编程、数据和模型基础打稳，再尽快走到应用和工程。

### 路线二：从零到模型原理

适合想把模型理解得更扎实的人：

`第零阶段 -> 第一阶段 -> 第二阶段 -> 第三阶段 -> 第四阶段 -> 第五阶段 -> 第六阶段 -> 第七阶段 -> 第八A阶段`

这条路线更强调视觉、NLP 和大模型原理的完整演进。

### 路线三：应用优先，再补理论

适合已经会一点 Python、想尽快做项目的人：

`第零阶段 -> 第一阶段 -> 第二阶段 -> 第四阶段 -> 第五阶段 -> 第八B阶段 -> 第九阶段`

之后再回补 `第三阶段 / 第七阶段 / 第八A阶段`。

## 怎么使用这套课最有效

### 1. 先抓主线，不要一开始横跳

更建议的方式是：

1. 按阶段往前推
2. 每学完一章就跑代码和做练习
3. 每完成一个阶段，至少做一次小总结
4. 每 2～3 个阶段做一个完整项目

### 2. 不要把所有内容都当“必修”

可以这样理解：

- 第零到第五阶段：主干基础
- 第六、七阶段：方向基础
- 第八 A / 第八 B / 第九阶段：大模型和 Agent 主线
- 第十阶段和选修模块：拓展视野与专项能力

### 3. 代码一定要亲手跑

这套课的目标不是“看懂教程”，而是让你能：

- 自己搭环境
- 自己调试
- 自己改例子
- 自己做最小闭环项目

## 学习过程中要记住的三件事

1. 不要等“全懂了”才开始做项目  
   小项目会倒逼你真正理解输入输出、评估和排障。

2. 不要把“追新”误当成“在进步”  
   前沿更新很快，但基础能力决定你能不能接住这些变化。

3. 不要怕回头复习  
   AI 学习本来就是循环式推进，很多内容都需要二刷和三刷。

祝你在这套课里真正建立起自己的 AI 全栈能力，而不是只攒下一堆名词。
