# AI 全栈学习教程

欢迎来到 AI 全栈学习教程！这是一份完整的从零基础到 AI Agent 开发的学习路径。

## 课程特点

- 📚 **系统完整**：从基础工具到前沿 AI 技术的完整学习链路
- 🚀 **实战导向**：每个阶段都包含实际项目和练习
- 💡 **循序渐进**：精心设计的学习难度曲线
- 🛠️ **工具全面**：涵盖现代 AI 开发所需的所有技术栈

## 课程结构

### 🎯 准备阶段
开始学习前的必要准备，包括学习策略、环境配置和职业规划。

### 📖 十大学习阶段
1. **第零阶段**：开发者工具基础（终端、Git、开发环境）
2. **第一阶段**：Python 编程基础
3. **第二阶段**：数据分析与可视化
4. **第三阶段**：数学基础（线性代数、概率统计、微积分）
5. **第四阶段**：机器学习基础
6. **第五阶段**：深度学习基础与 PyTorch
7. **第六阶段**：计算机视觉
8. **第七阶段**：自然语言处理
9. **第八阶段**：大语言模型（LLM）
   - 8A：LLM 核心原理与微调
   - 8B：LLM 应用开发与工程
10. **第九阶段**：AI Agent 开发
11. **第十阶段**：多模态与前沿技术

## 学习建议

- 💻 **边学边练**：每个章节都包含练习题和项目
- 🔄 **循环学习**：复习是学习的重要部分
- 🤝 **社区参与**：在 GitHub 上分享你的学习成果
