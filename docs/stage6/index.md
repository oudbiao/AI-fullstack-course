# 第六阶段：计算机视觉

| 信息 | 说明 |
|---|---|
| **预估学时** | 120～180 小时 |
| **前置要求** | 完成第五阶段 |

## 阶段概述

深入掌握计算机视觉核心任务与前沿技术

## 阶段导读

这一阶段建议把视觉任务按“粒度”来学：

1. 先看图像本身和基础处理
2. 再看整图分类
3. 再看框级检测
4. 最后看像素级分割和高级视觉任务

### 建议学习顺序

1. `ch01-cv-basics`
2. `ch02-classification`
3. `ch03-detection`
4. `ch04-segmentation`
5. `ch05-advanced`
6. `ch06-projects`

### 学这一阶段时最容易卡住的地方

- 把分类、检测、分割三类任务混在一起
- 只记模型名，不理解评估指标
- 只看样例图，不做错误分析

### 学完后的出口能力

- 能区分 CV 三大主线任务及其输入输出
- 能看懂 IoU、mAP、分割 IoU 这类核心指标
- 能完成一个最小视觉项目并做基本评估
