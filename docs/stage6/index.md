# 第六阶段：计算机视觉

| 信息 | 说明 |
|---|---|
| **预估学时** | 120～180 小时 |
| **前置要求** | 完成第五阶段 |

## 阶段概述

深入掌握计算机视觉核心任务与前沿技术

## 阶段导读

这一阶段建议把视觉任务按“粒度”来学：

1. 先看图像本身和基础处理
2. 再看整图分类
3. 再看框级检测
4. 最后看像素级分割和高级视觉任务

### 建议学习顺序

1. 第一章：视觉基础
2. 第二章：图像分类
3. 第三章：目标检测
4. 第四章：图像分割
5. 第五章：高级视觉
6. 第六章：项目实践

## 本阶段章节地图

| 章节 | 主题 | 主要解决什么问题 |
|---|---|---|
| 第一章 | 视觉基础 | 理解图像表示、像素、颜色、卷积和 OpenCV 基础 |
| 第二章 | 图像分类 | 学整图分类任务、迁移学习和分类训练 |
| 第三章 | 目标检测 | 学边界框、IoU、mAP 和检测模型主线 |
| 第四章 | 图像分割 | 学像素级预测、语义分割和实例分割 |
| 第五章 | 高级视觉 | 学 OCR、视频、人脸与 3D 等拓展方向 |
| 第六章 | 项目实践 | 把视觉任务真正做成可评估的小系统 |

### 学这一阶段时最容易卡住的地方

- 把分类、检测、分割三类任务混在一起
- 只记模型名，不理解评估指标
- 只看样例图，不做错误分析

## 适合怎么学这阶段

- 如果你是第一次学视觉，先以任务为主，不要以模型名为主
- 每学一个任务，都先搞清输入、输出和评估
- 做项目时，优先完成一个最小闭环，再追求更高指标

### 学完后的出口能力

- 能区分 CV 三大主线任务及其输入输出
- 能看懂 IoU、mAP、分割 IoU 这类核心指标
- 能完成一个最小视觉项目并做基本评估
