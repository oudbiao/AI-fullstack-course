# 学前导读：视觉基础这一章到底在学什么

这一章不是在学“几种图像 API”，而是在帮你建立视觉任务最基本的输入直觉。

## 零、先建立一张桥接线

如果你是从第五阶段 CNN 主线过来的，这一章最值得先看清的一件事是：

- 前面你已经知道卷积网络很适合图像
- 这一章开始回答“图像本身在计算机里到底长什么样”

所以这一章不是在偏离深度学习主线，而是在补：

> **视觉任务最基本的输入直觉。**

## 这一章的主线

```mermaid
flowchart LR
    A["图像是什么"] --> B["OpenCV 怎么读写和查看图像"]
    B --> C["基础图像处理"]
```

如果这一章没学稳，后面分类、检测、分割会很容易只剩模型名，没有输入感觉。

## 这一章更适合新人的学习顺序

1. 先看图像在计算机里到底是什么  
   把像素、通道、尺寸、颜色空间先看顺。

2. 再看 OpenCV 的读写和查看  
   先能把图读进来、看出来、拆通道。

3. 最后看基础处理  
   这时再做灰度、阈值、滤波之类操作，会更有感觉。

## 这一章最该先抓住什么

- 图像本质上是按空间组织起来的数字
- 通道和颜色空间会直接决定后面怎么处理图像
- 视觉模型之前，先要看懂“输入数据到底是什么”
