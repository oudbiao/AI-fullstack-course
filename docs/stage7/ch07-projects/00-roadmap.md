# 学前导读：综合项目这一章到底该怎么学

这一章不是继续堆模型，而是把前面学过的表示、分类、生成、抽取真正装进一个项目闭环。

## 三个项目分别在练什么

| 项目 | 你真正要练什么 |
|---|---|
| 智能问答系统 | 知识边界、检索、拒答和评估 |
| 文本摘要系统 | 生成结果的压缩质量和可解释性 |
| 信息抽取系统 | 从文本里稳定抽取结构化字段 |
