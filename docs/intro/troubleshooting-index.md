# 常见错误与排障索引

学习 AI 最常卡住的地方往往不是概念，而是环境、依赖、数据格式、接口、模型输入输出和工程边界。遇到问题时，先按下面的索引定位类型。

## 环境与工具

| 现象 | 优先检查 |
|---|---|
| 命令找不到 | 是否在正确目录，依赖是否安装，PATH 是否正确 |
| Python 包导入失败 | 是否激活环境，pip 是否装到同一个 Python |
| Git 推送失败 | 远程地址、权限、网络代理、分支名 |
| Docusaurus 显示旧内容 | 运行 `npm run clean`，清理 `.docusaurus` 和 `build` |

## Python 与数据分析

| 现象 | 优先检查 |
|---|---|
| 文件读不到 | 路径、当前工作目录、编码 |
| DataFrame 列不存在 | 列名空格、大小写、读取 header 是否正确 |
| 图表乱码 | 字体、中文显示配置、保存格式 |
| SQL 查询为空 | 表名、条件、连接库、数据是否已提交 |

## 机器学习与深度学习

| 现象 | 优先检查 |
|---|---|
| 分数异常高 | 是否数据泄漏，训练测试是否重复 |
| 模型不收敛 | 学习率、数据归一化、标签格式 |
| 训练好验证差 | 过拟合、数据少、增强不足 |
| GPU 不工作 | 驱动、CUDA、PyTorch 版本、设备选择 |

## LLM、RAG 与 Agent

| 现象 | 优先检查 |
|---|---|
| API 调用失败 | API Key、模型名、额度、网络、请求格式 |
| RAG 答非所问 | query、chunk、embedding、top-k、rerank |
| 引用不可信 | 是否记录来源，答案是否基于命中文档 |
| Agent 循环调用 | 停止条件、最大步数、工具返回是否清晰 |
| 工具参数错误 | schema、字段名、类型、必填项 |
| 成本突然升高 | 上下文过长、重试过多、循环、模型选择 |

## 排障记录模板

每次遇到问题，建议记录：错误现象、完整报错、复现步骤、你尝试过什么、最终解决方法、下次如何避免。这样的记录会慢慢变成你的个人工程经验库。
