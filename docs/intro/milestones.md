# Learning Milestones

![AI learning milestones timeline](/img/course/appendix-ai-milestones-timeline-en.png)

Milestones are not page-count goals. They answer one question:

> What can I now build, run, explain, and improve?

## Six Milestones

| Milestone | You have reached it when | Evidence |
| --- | --- | --- |
| 1. Starter | You can use the terminal, Git, and Python to run a small project | README, commit, command output |
| 2. Data Ready | You can clean a table, draw charts, and state limits | cleaned file, chart, data note |
| 3. Model Ready | You can train a baseline and explain metrics | baseline, metric table, error samples |
| 4. LLM App Ready | You can build a prompt or RAG workflow with checks | fixed tests, retrieval logs, citations |
| 5. Agent Ready | You can run a tool-using Agent with boundaries | trace, permission table, stop condition |
| 6. Portfolio Ready | Others can run and review your complete project | demo, README, evaluation report |

## How to Use This Page

After each stage, ask:

1. What artifact did I build?
2. What command reproduces it?
3. What evidence proves it worked?
4. What failed?
5. What will I improve next?

If the answers are clear, you have reached a milestone even if you did not read every optional page.
