# AI Learning Assistant Story Line

![AI learning assistant portfolio roadmap](/img/course/intro-project-portfolio-roadmap-en.png)

This page is optional. Use it if a story helps you stay motivated.

The idea is simple: the whole course can grow one project, an **AI learning assistant**. At first it is only a folder. Later it can save notes, analyze records, call models, search course materials, use tools, and become a demo-ready product.

## Four Acts

| Act | Assistant growth | Course focus |
| --- | --- | --- |
| 1 | From empty folder to learning notebook | Tools, Python, data analysis |
| 2 | Learns to judge patterns and risk | Math, machine learning, deep learning |
| 3 | Can communicate, retrieve, and act | Prompt, RAG, Agent |
| 4 | Becomes a portfolio product | CV, NLP, multimodal, capstone |

Do not build all features at once. Each stage unlocks one small ability.

## Stage Story Cards

| Stage | Ability unlocked | Evidence to keep |
| --- | --- | --- |
| 1 Tools | The assistant has a project workspace | README, first commit, run screenshot |
| 2 Python | It can save learning tasks | `tasks.json`, CLI output, error handling |
| 3 Data | It can summarize learning records | chart, cleaned data, quality note |
| 4 Math | It understands similarity, probability, and change | formula-to-code mini experiment |
| 5 Machine Learning | It predicts or classifies learning risk | baseline metrics, error samples |
| 6 Deep Learning | It reads training signals | loss curve, config, failed run note |
| 7 LLM/Prompt | It writes plans or review cards | prompt versions, fixed test cases |
| 8 RAG | It answers from course materials | retrieval logs, citation checks |
| 9 Agent | It breaks down a task and calls tools | tool trace, permissions, stop condition |
| 10-12 Extensions | It handles one specialized medium | before/after examples and evaluation |
| Graduation | It becomes a demoable product | demo, README, evaluation report |

## One README Note per Stage

After a stage, add a short entry:

```md
## Stage 2: Learning Notebook

Unlocked ability:
The assistant can add, view, and complete learning tasks.

How to run:
python main.py add "Learn Python file I/O"

Evidence:
tasks.json was generated and can be loaded again.

One failure:
The program crashed when tasks.json was broken.

Fix or next step:
Handle JSONDecodeError and show a friendly message.
```

This is enough. The story is not extra work; it is a simple way to connect stage outputs into one portfolio thread.
