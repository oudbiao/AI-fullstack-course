# 0.4 Recommended Learning Path

![Recommended Learning Path Selection Diagram](/img/course/intro-learning-path-selection-en.webp)

If you are unsure, choose the beginner path: **Chapter 1 -> Chapter 9 in order, one small output per stage.**

| Your goal | First route |
|---|---|
| I am new | Follow chapters 1-9 |
| I already code | Skim 1-6, focus on 7-9 |
| I need a portfolio | Keep README, screenshots, logs, failures |
| I care about models | Spend more time on math, ML, DL, Transformer |

Do not switch routes every week. Read briefly, run something, keep evidence, then continue to [Chapter 1](/ch01-tools).
