## QA Result

All six accepted candidates are native vertical teaching images at `1024x1792`.

Visual checks passed:

- Large, readable title and section labels for mobile reading.
- Classroom whiteboard / notebook teaching style, not a marketing poster.
- Sparse text with four main workflow/evidence blocks.
- No tiny terminal output, endpoint lists, version strings, dates, capacities, benchmark scores, secrets, citations, or fake metrics.
- No real provider logos, animal mascots, watermarks, or brand-like runtime icons.
- The English, Simplified Chinese, and Japanese versions keep the same teaching purpose and visual rhythm.

Rejected earlier candidates:

- Dense poster-like versions with too many small labels and internal file/checklist text.
- Candidates that invented version/date/capacity-style details.
- Partial generations from unstable API runs that did not complete the trilingual set.
