# 学習ルートFAQ

![FAQ意思決定ツリー](/img/course/appendix-faq-decision-tree-ja.png)

このページは、迷いを減らして前に進むためのものです。多くの質問は、長く計画するより小さなプロジェクトで答えたほうが早いです。

## すぐ分かる答え

| 質問 | 短い答え |
|---|---|
| 完全初心者でも学べる？ | はい。ツール、Python、データから始めます。 |
| Python が分かるなら飛ばせる？ | 速く読めますが、ステージプロジェクトの確認はしてください。 |
| RAG や Agent だけやりたい。数学は必要？ | ベクトル、類似度、指標、評価の直感は必要です。 |
| フロントエンドは必要？ | 初回は不要です。CLI、Notebook、Streamlit、FastAPI で始められます。 |
| GPU は必要？ | 初回は不要です。API と小さなローカル例で多くを学べます。 |
| どのくらいかかる？ | 4週間は短期集中、8週間はポートフォリオ形作り、12週間は安定した一周です。 |
| 理論とプロジェクト、どちらが先？ | 最小プロジェクトを動かし、その問題意識で理論を読みます。 |
| 各ステージでプロジェクトが必要？ | 大きなステージごとに小さな動く成果を1つ残します。 |
| 卒業制作はいつ始める？ | 最小の RAG または Agent ループが動いたら始められます。 |
| 詰まったら？ | ルート変更の前に、1つのコマンド、1つのファイル、1つの入力まで絞ります。 |

## 標準ルート

迷うならこの順番です。

1. クイック体験。
2. 環境準備。
3. ステージ1-3でツール、Python、データ。
4. ステージ4-6でモデルの直感。
5. ステージ7-9で LLM、RAG、Agent。
6. ステージ10-12またはデプロイ選修から卒業方向を1つ選ぶ。

## 詰まったとき

コース全体を最初からやり直さないでください。現在のステージ課題に戻り、最小コマンドで再現し、トラブルシューティング索引を確認し、修正をプロジェクトメモに残します。
