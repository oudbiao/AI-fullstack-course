# 全コースプロジェクトマトリクス

![プロジェクトポートフォリオロードマップ](/img/course/intro-project-portfolio-roadmap-ja.png)

このページは、「各ステージで何を作るか」だけに答えます。初回は小さく作り、見せたいプロジェクトだけをポートフォリオ用に強化します。

## 1. ステージ別プロジェクト地図

| ステージ | 最小プロジェクト | 残す証拠 |
|---|---|---|
| 1 ツール | リポジトリを作り Python を動かす | README、コマンド記録、スクリーンショット |
| 2 Python | CLI タスク管理または簡単な API | 実行コマンド、入出力例 |
| 3 データ | 1つの CSV を分析して図を作る | クリーニング記録、図、結論 |
| 4 AI 数学 | ベクトル、確率、勾配の小実験 | 図解、小さなコード、説明 |
| 5 機械学習 | 分類または回帰の baseline | 指標、baseline、エラー例 |
| 6 深層学習 | PyTorch 学習ループ | 設定、曲線、失敗例 |
| 7 LLM と Prompt | Prompt アシスタントまたは復習カード生成 | Prompt 版、出力比較 |
| 8 RAG | 引用付き Markdown Q&A | 質問セット、検索ログ、引用チェック |
| 9 Agent | ツール呼び出し型の計画 Agent | ツール schema、実行 trace、制限 |
| 10 Vision | 分類、OCR、検出の実験 | ラベル、指標、可視化結果 |
| 11 NLP | テキスト分類または情報抽出 | ラベル規則、指標、エラー例 |
| 12 マルチモーダル | 画像、音声、動画、または複合ワークフロー | 元素材、生成ログ、確認ルール |

## 2. 強化ルール

| 版 | 目的 | 追加する証拠 |
|---|---|---|
| 最小版 | 動くことを示す | README、コマンド、出力例 |
| 標準版 | 他の人が再現できることを示す | 依存関係、設定メモ、ログ |
| ポートフォリオ版 | 評価でき、説明できることを示す | 評価セット、失敗例、画像、振り返り |

チャレンジ課題は必須ではありません。最初の動く流れを作る前ではなく、主ルートが安定した後に使います。

## 3. おすすめリポジトリ構成

```text
ai-fullstack-portfolio/
├── ch01-tools/
├── ch02-python-cli/
├── ch03-data-report/
├── ch05-ml-baseline/
├── ch08-rag-assistant/
├── ch09-agent-planner/
└── final-ai-app/
```

各フォルダには、README、ソースコード、サンプル入力またはデータ、出力の証拠、失敗サンプル、次の一手を入れます。
