# 学習マイルストーン

![AI 学習マイルストーンタイムライン](/img/course/appendix-ai-milestones-timeline-ja.png)

マイルストーンは「何ページ読んだか」ではありません。次の質問に答えるものです。

> 今、自分は何を作り、動かし、説明し、改善できるか？

## 6 つのマイルストーン

| マイルストーン | 到達したと言える状態 | 証拠 |
| --- | --- | --- |
| 1. スタート | ターミナル、Git、Python で小さなプロジェクトを動かせる | README、commit、コマンド出力 |
| 2. データ準備完了 | 表を整理し、グラフを作り、限界を説明できる | 整理済みファイル、グラフ、データメモ |
| 3. モデル準備完了 | baseline を学習し、指標を説明できる | baseline、指標表、エラーサンプル |
| 4. LLM アプリ準備完了 | チェック付きの Prompt または RAG 流れを作れる | 固定テスト、検索ログ、引用 |
| 5. Agent 準備完了 | 境界のあるツール利用 Agent を動かせる | trace、権限表、停止条件 |
| 6. ポートフォリオ準備完了 | 他の人が完全なプロジェクトを実行・確認できる | Demo、README、評価レポート |

## このページの使い方

各ステージの後で自分に聞きます。

1. どんな成果物を作ったか？
2. どのコマンドで再現できるか？
3. 有効だと分かる証拠は何か？
4. どこで失敗したか？
5. 次に何を改善するか？

答えがはっきりしていれば、すべての任意ページを読んでいなくても、そのマイルストーンには到達しています。
