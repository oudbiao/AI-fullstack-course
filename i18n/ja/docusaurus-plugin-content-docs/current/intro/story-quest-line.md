# AI 学習アシスタントのストーリーライン

![AI 学習アシスタントのポートフォリオロードマップ](/img/course/intro-project-portfolio-roadmap-ja.png)

このページは任意です。物語があるほうが続けやすい人だけ使ってください。

考え方はシンプルです。コース全体で 1 つの **AI 学習アシスタント** を育てます。最初はただのフォルダですが、後でノートを保存し、学習記録を分析し、モデルを呼び出し、コース資料を検索し、ツールを使い、最後はデモできる作品になります。

## 4 つの幕

| 幕 | アシスタントの成長 | コースの重点 |
| --- | --- | --- |
| 第一幕 | 空フォルダから学習ノートになる | ツール、Python、データ分析 |
| 第二幕 | パターンやリスクを判断する | 数学、機械学習、深層学習 |
| 第三幕 | 会話し、検索し、行動する | Prompt、RAG、Agent |
| 第四幕 | ポートフォリオ作品になる | CV、NLP、マルチモーダル、卒業制作 |

すべての機能を一度に作る必要はありません。各ステージで小さな能力を 1 つだけ解放します。

## ステージ別ストーリーカード

| ステージ | 解放する能力 | 残す証拠 |
| --- | --- | --- |
| 1 ツール | アシスタントに作業場所ができる | README、最初の commit、実行スクリーンショット |
| 2 Python | 学習タスクを保存できる | `tasks.json`、CLI 出力、エラー処理 |
| 3 データ | 学習記録を集計できる | グラフ、整理済みデータ、品質メモ |
| 4 数学 | 類似、確率、変化を理解する | 数式からコードへの小実験 |
| 5 機械学習 | 学習リスクを予測または分類する | baseline 指標、エラーサンプル |
| 6 深層学習 | 学習の信号を読む | loss カーブ、設定、失敗記録 |
| 7 LLM/Prompt | 計画や振り返りカードを書く | Prompt バージョン、固定テストケース |
| 8 RAG | コース資料に基づいて答える | 検索ログ、引用チェック |
| 9 Agent | タスクを分解し、ツールを呼ぶ | ツール trace、権限、停止条件 |
| 10-12 発展 | 1 つの専門メディアを扱う | 前後比較例と評価 |
| 卒業制作 | デモできるプロダクトになる | Demo、README、評価レポート |

## 各ステージで README に 1 件書く

ステージが終わったら、短い記録を追加します。

```md
## 第 2 ステージ：学習ノート

解放した能力：
アシスタントは学習タスクを追加、表示、完了できます。

実行方法：
python main.py add "Python のファイル入出力を学ぶ"

証拠：
tasks.json が生成され、再読み込みできました。

失敗 1 件：
tasks.json が壊れているとプログラムが落ちました。

修正または次の一歩：
JSONDecodeError を処理し、分かりやすいメッセージを出します。
```

これで十分です。ストーリーラインは追加作業ではなく、各ステージの成果を 1 本のポートフォリオ線につなぐための短い見方です。
