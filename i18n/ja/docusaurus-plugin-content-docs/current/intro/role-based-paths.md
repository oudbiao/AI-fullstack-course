# 目標別の学習ルート

![役割別学習ルート選択マップ](/img/course/intro-role-based-paths-map-ja.png)

まだ迷うなら、まずステージ1-9のメインルートを進めてください。このページは、初回後にどこを深く読むか決めるために使います。

## 30秒で選ぶ

| 目標 | 深く読む | 初回は流せる | ポートフォリオ例 |
|---|---|---|---|
| AIアプリケーション | 1、2、3、7、8、9 | 4-6の細部 | API、RAG、ログ、デプロイメモ付きAIアプリ |
| RAG | 2、3、5、7、8 | 高度なAgent | 検索ログと引用評価がある知識ベース |
| Agent | 2、3、7、8、9 | 高度な学習 | trace と権限境界を持つツール呼び出しAgent |
| モデル | 4、5、6、7、8のモデルデプロイ | フロントエンド細部 | モデル比較、微調整、推論レポート |
| ポートフォリオ | 1-9、その後で卒業方向を1つ | ニッチな拡張 | README、デモ、失敗サンプルがある作品ストーリー |

## 最小プロジェクトセット

| ルート | まず作るもの |
|---|---|
| アプリ | Python API + Prompt アシスタント + RAG Q&A + 簡単なデプロイ |
| RAG | データ整形 + 文書分割デモ + RAG Q&A + 評価セット |
| Agent | 関数呼び出しデモ + 複数手順Agent + 安全確認例 |
| モデル | ML baseline + DL学習実験 + モデル比較 |
| ポートフォリオ | データレポート + ML baseline + RAG助手 + Agent助手 + 卒業制作 |

## 切り替えルール

毎週ルートを変えないでください。まず小さなプロジェクトを1つ終え、何が難しかったかを振り返り、それから RAG、Agent、モデル、プロダクト提出のどれを深めるか決めます。
