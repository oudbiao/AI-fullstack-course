# AI フルスタック能力マップ

![AI フルスタック能力マップ](/img/course/intro-ai-fullstack-capability-map-ja.png)

まず図を見ます。コースは1本の道です。

```text
tools -> data -> models -> LLM -> RAG -> Agent -> delivery
```

今は細部を全部理解しなくて大丈夫です。

| 詰まっていること | 戻る場所 |
|---|---|
| コードが動かない | ツールと Python |
| 入力が乱れている | データ |
| 答えが信頼できない | 評価と RAG |
| 行動が制御できない | Agent trace と権限 |

次に学習ルートを選びます。
