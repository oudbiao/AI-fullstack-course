# 学習戦略

![継続学習フライホイール](/img/course/appendix-continuous-learning-flywheel-ja.png)

コース全体で 1 つの学習ループを使います。

> 考え方を見る -> 最小例を動かす -> 小さな成果物を作る -> 証拠を記録する -> 次の一歩を振り返る。

## 3 つのルール

| ルール | 防ぐもの | 最小行動 |
| --- | --- | --- |
| プロジェクトで学ぶ | チュートリアルを見ても証拠が残らない | 各ステージで動く成果物を 1 つ残す |
| 理論を実践の近くで学ぶ | 数学やモデル理論だけを何週間も読む | 1 つの直感に 1 つの小実験を添える |
| 証拠で学ぶ | 何をやったか忘れてしまう | コマンド、出力、エラー、指標を保存する |

## 週間リズム

| 時間ボックス | 行動 |
| --- | --- |
| 25 分 | 小さな節を読む、または図を見る |
| 25 分 | 例を実行または変更する |
| 10 分 | README メモか失敗メモを書く |
| 週末 | 作品集証拠として残す結果を 1 つ選ぶ |

## 避けたい落とし穴

| 落とし穴 | よりよい動き |
| --- | --- |
| チュートリアル収集 | 次の資料を開く前に 1 つ例を動かす |
| 完璧主義 | まず不格好でも動く版を作る |
| ルートを頻繁に変える | 今のステージチェックポイントを終える |
| 理論だけ学ぶ | 小さな実験を足す |
| Demo だけ作る | 評価または失敗例を足す |
