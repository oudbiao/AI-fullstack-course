# 2.0 学習ガイドとタスクシート：Python プログラミング基礎

このページを第 2 章のコントロールパネルとして使ってください。目標はすべての構文を暗記することではありません。動き、データを保存し、エラーに対応でき、説明できる小さな Python プログラムを書くことです。

## 2.0.1 推奨学習順序

![Python 学習ガイド最小閉ループ](/img/course/ch02-study-guide-program-loop-ja.png)

まずこの流れを覚えます：**入力 -> データ構造 -> 関数 -> ファイルまたは出力**。

| 順番 | セクション | 重点 | 残す証拠 |
|---|---|---|---|
| 1 | `2.1 Python 言語基礎` | 変数、型、演算子、入出力、制御フロー | 手で入力して変更した小さなスクリプト 5 個 |
| 2 | `2.1 データ構造と関数` | list、dict、関数、モジュール | 小さな関数に分けたスクリプト |
| 3 | `2.2 Python 応用` | クラス、例外、ファイル、イテレータ、型ヒント | ファイル読み込みまたは JSON 保存の例 |
| 4 | `2.3 ステージプロジェクト` | CLI、クローラー、Web API、AI API 体験 | 実行できるプロジェクトフォルダ |
| 5 | `2.5 ハンズオンワークショップ` | CLI コマンド、JSON 永続化、統計、レポート出力 | `ch02_output/` と README メモ |

最初から高度な構文を追わなくて大丈夫です。まず動かし、何かを保存し、読み戻し、各関数がなぜ必要かを説明できるようにしましょう。

## 2.0.2 始める前に知っておく用語

| 用語 | 英語の正式名称 | この章での最初の意味 |
|---|---|---|
| `CLI` | Command-Line Interface | 文字コマンドで操作するプログラム。 |
| `I/O` | Input/Output | データがプログラムに入り、結果が外へ出ること。 |
| `JSON` | JavaScript Object Notation | 入れ子のデータを保存しやすい軽量テキスト形式。 |
| `API` | Application Programming Interface | あるプログラムが別のプログラムを呼ぶための入口。 |
| `SDK` | Software Development Kit | API を使いやすい関数として包んだライブラリ。 |

略語は能力として読みましょう。インターフェース、保存、リクエスト、レスポンス、ワークフローのどれに近いかを見ると迷いにくくなります。

## 2.0.3 この段階で必ず完了するタスク

| タスク | 成果物 | クリア基準 |
|---|---|---|
| 最小のプログラムループを身につける | 小さなスクリプト 5 個 | 変数、条件分岐、ループ、関数で簡単な課題を解ける |
| データを整理する | list/dict 練習ファイル | list、dict、set、tuple の使い分けを説明できる |
| 関数に分ける | 複数関数を持つ小さなプログラム | 各関数の入力、出力、責務が明確 |
| ファイルを読み書きする | テキスト、JSON、JSONL の例 | パスを推測せず、保存して読み戻せる |
| エラーを扱う | デバッグメモ | 行番号を見つけ、変数値を確認し、よくあるエラーを 2 つ以上直せる |
| ハンズオンを完了する | `ch02_output/` | コマンド実行、JSON 保存、完了処理、統計表示、レポート出力ができる |

## 2.0.4 ミニプロジェクトの型

この段階の Python プロジェクトは、同じ小さな型で進めます。

1. 実行コマンドを書く。
2. とても小さな入力を用意する。
3. 分かりやすい出力を表示または保存する。
4. エラーケースを 1 つ追加する。
5. README に何を変えたかを記録する。

この型は、タスク管理ツール、クローラー、Web API、AI API デモにそのまま使えます。

## 2.0.6 段階通過質問

第 3 章に進む前に、次の質問に答えられるか確認してください。

- 繰り返しコードはいつ関数にするべきか？
- 辞書がリストより向いているのはどんなときか？
- ファイルパスはどのフォルダを基準に計算されるか？
- `print` と `return` は何が違うか？
- 1 つのスクリプトは、どうやってモジュール、データ、README を持つ小さなプロジェクトになるか？

コマンドラインのタスク管理ツールをゼロから作り、検索、並べ替え、分類、エクスポートのような小さな機能を 1 つ追加できれば、次へ進んで大丈夫です。
