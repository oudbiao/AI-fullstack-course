# 3.0 学习指南与任务单：数据分析与可视化

把这一页当成第 3 章的控制台。目标不是背完每个 Pandas 方法，而是把原始数据变成别人能信任的结论。

## 3.0.1 按这个顺序学习

![数据分析学习指南最小闭环](/img/course/ch03-study-guide-data-loop-vertical.png)

先记住一条工作流：**读取 -> 检查 -> 清洗 -> 汇总 -> 可视化 -> 解释**。

| 顺序 | 小节 | 重点 | 需要留下的证据 |
|---|---|---|---|
| 1 | `3.1 从 Python 到数据分析` | 为什么纯 Python 处理表格会变痛苦 | 一份前后对比小笔记 |
| 2 | `3.2 NumPy 科学计算` | 数组、shape、广播、向量化 | 一个数组练习文件 |
| 3 | `3.3 Pandas 数据处理` | DataFrame、选择、清洗、groupby、merge | 一份清洗后的表和清洗日志 |
| 4 | `3.4 数据可视化` | 图表选择、标签、结论、限制 | 3 张对应 3 个问题的图 |
| 5 | `3.5 数据库基础` | SQL 与关系型数据，选修但很有用 | 一个查询或 join 示例 |
| 6 | `3.6 阶段项目` | EDA 报告和可复现流水线 | 报告、图表文件和工作坊输出 |

不要先画图。先理解字段、单位、缺失值和样本来源。

## 3.0.2 开始前需要认识的术语

| 术语 | 本章先怎么理解 |
|---|---|
| `CSV` | 纯文本表格，每一行是一条记录。 |
| `JSON` | API 经常返回的嵌套数据格式。 |
| `DataFrame` | Pandas 表格，有行、列、列名和索引。 |
| `Series` | DataFrame 的一列。 |
| `dtype` | 某一列或数组的数据类型。 |
| `EDA` | 探索性数据分析，建模前的第一轮摸底。 |
| `groupby` | 按类别拆分，计算统计量，再合并结果。 |
| `merge` / `join` | 用共同键把表合并起来。 |
| `vectorization` | 让 NumPy/Pandas 一次处理很多值。 |

如果 API 记不住，先把它翻译回工作流步骤：读取、检查、清洗、转换、可视化或解释。

## 3.0.4 图表选择规则

画图前，先写问题。

| 问题 | 优先尝试的图表 |
|---|---|
| 哪个类别最大？ | 柱状图 |
| 数值随时间怎么变化？ | 折线图 |
| 两个数值是否相关？ | 散点图 |
| 分布长什么样？ | 直方图或箱线图 |
| 不同组有什么差异？ | 分组柱状图或箱线图 |

没有问题的图表只是装饰；有问题的图表才是证据。

## 3.0.6 阶段检查点

进入第 4 章前，确认自己能回答这些问题：

- 为什么数组的 `shape` 很重要？
- Series 和 DataFrame 有什么区别？
- 缺失值如何改变结论？
- 每张图到底回答了什么问题？
- 为什么数据分析报告必须写限制？

只要你能把一个 CSV 从读取、清洗、分析、可视化做到一段简短结论，就可以继续往后学。
