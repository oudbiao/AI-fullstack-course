## 2.0.2 练习清单

| 检查项 | 证据 |
|---|---|
| 能运行 5 个包含变量、条件、循环的小脚本 | `practice/` 文件夹 |
| 能在列表、字典、元组、集合之间做选择 | 一段数据结构说明 |
| 能把重复代码拆成函数 | 重构后的脚本 |
| 能保存并重新读取 JSON | `tasks.json` |
| 能处理一次文件损坏或输入错误 | 调试记录 |
| 能完成跟做工作坊 | `ch02_output/` |
