# 按目标选择学习路线

![按角色选择学习路线图](/img/course/intro-role-based-paths-map.png)

如果还不确定目标，先按第 1-9 阶段主线学习。这一页只用来决定第一遍之后哪里需要深读。

## 30 秒选择

| 目标 | 深读 | 第一遍可快读 | 作品集项目 |
|---|---|---|---|
| AI 应用工程 | 1、2、3、7、8、9 | 4-6 的细节 | 带 API、RAG、日志和部署说明的 AI 应用 |
| RAG 工程 | 2、3、5、7、8 | 高级 Agent | 有检索日志和引用评估的知识库 |
| Agent 开发 | 2、3、7、8、9 | 高级训练 | 带 trace 和权限边界的工具调用 Agent |
| 模型工程 | 4、5、6、7、8 的模型部署 | 前端细节 | 模型对比、微调或推理报告 |
| 求职作品集 | 1-9，再选一个毕业方向 | 小众扩展 | 有 README、演示和失败样本的项目故事 |

## 最小项目组合

| 路线 | 先做这些 |
|---|---|
| 应用 | Python API + Prompt 助手 + RAG 问答 + 简单部署 |
| RAG | 数据清洗 + 文档切分 Demo + RAG 问答 + 评估集 |
| Agent | 函数调用 Demo + 多步 Agent + 安全确认示例 |
| 模型 | ML baseline + DL 训练实验 + 模型对比 |
| 作品集 | 数据报告 + ML baseline + RAG 助手 + Agent 助手 + 毕业项目 |

## 切换规则

不要每周换一个方向。先完成一个小项目，复盘哪里最难，再决定深入 RAG、Agent、模型还是产品交付。
