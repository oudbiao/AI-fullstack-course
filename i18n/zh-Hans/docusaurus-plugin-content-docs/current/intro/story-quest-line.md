# AI 学习助手剧情线

![AI 学习助手作品集路线图](/img/course/intro-project-portfolio-roadmap.png)

这页是可选的。如果故事能帮你保持动力，就用它。

思路很简单：整门课可以围绕一个项目成长，也就是 **AI 学习助手**。一开始它只是一个文件夹；后来它能保存笔记、分析学习记录、调用模型、检索课程资料、使用工具，最后变成可展示的作品。

## 四幕剧情

| 幕 | 助手如何成长 | 对应课程重点 |
| --- | --- | --- |
| 第一幕 | 从空文件夹变成学习记事本 | 工具、Python、数据分析 |
| 第二幕 | 学会判断模式和风险 | 数学、机器学习、深度学习 |
| 第三幕 | 会交流、会检索、会行动 | Prompt、RAG、Agent |
| 第四幕 | 变成作品集项目 | CV、NLP、多模态、毕业项目 |

不要一次做完所有功能。每个阶段只解锁一个小能力。

## 阶段剧情卡

| 阶段 | 解锁能力 | 保留证据 |
| --- | --- | --- |
| 1 工具 | 助手有了项目工作台 | README、第一次 commit、运行截图 |
| 2 Python | 助手能保存学习任务 | `tasks.json`、CLI 输出、错误处理 |
| 3 数据 | 助手能汇总学习记录 | 图表、清洗后数据、质量说明 |
| 4 数学 | 助手理解相似、概率和变化 | 公式到代码的小实验 |
| 5 机器学习 | 助手能预测或分类学习风险 | baseline 指标、错误样本 |
| 6 深度学习 | 助手能读懂训练信号 | loss 曲线、配置、失败记录 |
| 7 LLM/Prompt | 助手能写计划或复盘卡 | Prompt 版本、固定测试样例 |
| 8 RAG | 助手能基于课程资料回答 | 检索日志、引用检查 |
| 9 Agent | 助手能拆任务并调用工具 | 工具 trace、权限、停止条件 |
| 10-12 拓展 | 助手处理一个专项媒介 | 前后对比样例与评估 |
| 毕业项目 | 助手成为可演示产品 | Demo、README、评估报告 |

## 每阶段写一条 README 记录

完成阶段后，补一段短记录：

```md
## 第 2 阶段：学习记事本

解锁能力：
助手可以新增、查看和完成学习任务。

运行方式：
python main.py add "学习 Python 文件读写"

成功证据：
生成了 tasks.json，并能再次读取。

一个失败：
tasks.json 损坏时程序崩溃。

修复或下一步：
处理 JSONDecodeError，并给出友好提示。
```

这就够了。剧情线不是额外作业，而是把每阶段产出串成一条作品集主线。
