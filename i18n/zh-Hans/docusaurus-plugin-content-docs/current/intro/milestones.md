# 学习里程碑

![AI 学习里程碑时间线](/img/course/appendix-ai-milestones-timeline.png)

里程碑不是“看完多少页”。它只回答一个问题：

> 我现在能做出、运行、解释并改进什么？

## 六个里程碑

| 里程碑 | 什么时候算到达 | 证据 |
| --- | --- | --- |
| 1. 起步 | 能用终端、Git 和 Python 跑一个小项目 | README、commit、命令输出 |
| 2. 数据就绪 | 能清洗表格、画图并说明限制 | 清洗后文件、图表、数据说明 |
| 3. 模型就绪 | 能训练 baseline 并解释指标 | baseline、指标表、错误样本 |
| 4. LLM 应用就绪 | 能构建带检查的 Prompt 或 RAG 流程 | 固定测试、检索日志、引用 |
| 5. Agent 就绪 | 能运行有边界的工具型 Agent | trace、权限表、停止条件 |
| 6. 作品集就绪 | 别人能运行并审查你的完整项目 | Demo、README、评估报告 |

## 怎么用这页

每个阶段结束后问自己：

1. 我做出了什么成果物？
2. 用什么命令能复现？
3. 什么证据证明它有效？
4. 哪里失败过？
5. 下一步要改进什么？

如果答案清楚，就算没有读完所有选修页，你也已经到达了一个里程碑。
