# 学习路线常见问题

![FAQ 决策树](/img/course/appendix-faq-decision-tree.png)

这一页用来快速做决定，然后继续前进。很多问题，与其反复规划，不如用一个小项目回答。

## 快速答案

| 问题 | 简短回答 |
|---|---|
| 完全新手能学吗？ | 能。先从工具、Python 和数据开始。 |
| 已经会 Python 能跳过吗？ | 可以快读，但阶段项目检查仍要做。 |
| 只想做 RAG 或 Agent，需要数学吗？ | 需要足够直觉，尤其是向量、相似度、指标和评估。 |
| 需要会前端吗？ | 第一遍不需要。CLI、Notebook、Streamlit 或 FastAPI 足够起步。 |
| 需要 GPU 吗？ | 第一遍不需要。API 和小型本地例子能覆盖多数阶段。 |
| 多久能学完？ | 4 周适合快跑，8 周适合作品集雏形，12 周适合稳步完整走完。 |
| 先理论还是先项目？ | 先跑最小项目，再带着具体问题读理论。 |
| 每阶段都要做项目吗？ | 每个大阶段至少留下一个小的可运行产物。 |
| 什么时候开始毕业项目？ | 最小 RAG 或 Agent 闭环跑通后就可以开始。 |
| 卡住怎么办？ | 换路线前，先把问题缩小到一个命令、一个文件或一个输入。 |

## 默认路线

如果还不确定，就按这个顺序：

1. 快速体验。
2. 环境准备。
3. 第 1-3 阶段：工具、Python、数据。
4. 第 4-6 阶段：建立足够模型直觉。
5. 第 7-9 阶段：LLM、RAG、Agent。
6. 从第 10-12 阶段或部署选修里选择一个毕业方向。

## 卡住时

不要重开整套课程。回到当前阶段任务，用最小命令复现问题，查排障索引，并把修复过程写进项目笔记。
