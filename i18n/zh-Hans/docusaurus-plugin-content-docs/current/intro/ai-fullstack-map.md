# 0.3 AI 全栈能力地图

![AI 全栈能力地图](/img/course/intro-ai-fullstack-capability-map.png)

先看图。课程是一条路线：

```text
tools -> data -> models -> LLM -> RAG -> Agent -> delivery
```

现在不需要懂每个细节，只记住：

| 如果卡在... | 回到... |
|---|---|
| 代码跑不起来 | 工具和 Python |
| 输入很乱 | 数据 |
| 回答不可靠 | 评估和 RAG |
| 动作不可控 | Agent trace 和权限 |

可选背景：如果想知道这些能力是怎样发展出来的，可以快速看一眼 [AI 15 阶段发展史](/appendix/ai-milestones)。

下一步，选择学习路线。
